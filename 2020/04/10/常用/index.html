1 Python随机生成的几种方法

各种随机：

import random
random.uniform(10,20) #random generate [10,20] 的 float
random.randint(20,30) #random generate [20,30] 的 整数
random.randrange(0,101,2) #随机选取0到100间的偶数
random.choice('abcdefg&#%^*f') #随机字符
random.choice ( ['apple','pear', 'peach', 'orange', 'lemon'] ) #随机选取字符串

洗牌： 

items = [1, 2, 3, 4, 5, 6]  
random.shuffle(items)

2 保留有效数位

字符串

保留有效数字：

 format(12.456789, '.3g')
'12.5'

小数点后n位：

`format(12.456789,'.3f')

科学计数法：

format(12.456789, '.3e')
'1.246e+01'

换成百分比之后保留小数点后n位数：

format(12.456789, '.3%')
'1245.679%'

浮点数：

float(format(12.456789, '.3g'))
